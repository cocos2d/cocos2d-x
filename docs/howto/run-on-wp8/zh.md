Title: 在 Windows Phone 8 上运行

在 Windows Phone 8 上运行
=======================

由于 WP8 的特殊性，有以下注意事项：

-   **如何查看 log 输出和 debug 调试**

    以 helloworld 工程为例：

    1.  右键查看 helloworld 属性 (不是 helloworldComponent)
    2.  选择 Debug 标签
    3.  UI Task: 选择 "Native Only"

~

## 不支持的特性

-   Lua File System Extension
-   Lua Socket Extension
-   MP3 Decode
